# stand-alone-web

> 这里是项目描述信息，请用一句话描述这个项目吧！！！

### 快速开始

```shell
# 第一步：克隆项目
git clone -b stand-alone-web https://github.com/code-chic/web.git 
# 第二步：切换到项目文件夹
cd web
# 第三步：运行项目
npm run serve
```



### 其他指令

```shell
npm run buld # 构建生产应用
npm run generate # 自动生成模块
npm run archive # 代码打包（压缩包哦^_^!）
```
